# ターゲット企業リストアップガイド

中小派遣会社（登録スタッフ50〜300名規模）のリスト構築方法。
**すべて無料・合法・公開情報**ベースです。

---

## 🎯 ターゲット企業の理想像

| 項目 | 詳細 |
|---|---|
| 業種 | 製造系・物流系・食品加工系・軽作業系の派遣 |
| 規模 | 社員数5〜30名、登録スタッフ50〜300名 |
| エリア | 愛知・岐阜・三重（製造系）／関東・関西（物流系）／全国（介護系） |
| 決裁者 | 社長または営業部長（即決傾向あり） |
| IT成熟度 | 低〜中（紙タイムカード or 個別のExcel管理が多い） |

## 📋 リストアップ方法 5選（無料）

### 方法1：厚労省「人材サービス総合サイト」⭐⭐⭐⭐⭐

**URL：** https://jinzai-sougou.mhlw.go.jp/

**特徴：**
- **派遣業許可事業者の全リストが完全公開**
- 都道府県・業種・規模で絞り込み可能
- 法令遵守の派遣会社のみが掲載されている

**使い方：**
1. サイトを開く
2. 「事業所を検索」をクリック
3. 「都道府県」を選択（例：愛知県）
4. 「労働者派遣事業」にチェック
5. 検索結果から「会社名・所在地・電話番号」を取得
6. Excelに転記してリスト化

**取得可能な情報：**
- 事業所名・所在地・電話番号・許可番号
- 派遣事業の業種分類

**おすすめのフィルター方法：**
- 全国13,000社中、「製造系派遣」で絞る → 約2,000社
- 「中小派遣会社」のうち「社員5〜30名」推定 → 約500〜800社

### 方法2：Google検索 + マップ ⭐⭐⭐⭐

**検索クエリ例：**

```
「○○県 派遣会社」
「○○市 製造派遣」
「○○エリア 軽作業派遣」
「派遣会社 中小 ○○県」
```

**取得方法：**
1. Google マップで地域指定して検索
2. 会社一覧から HP・電話番号を取得
3. 会社規模（社員数）は HP の「会社概要」で確認

**メリット：** 地域密着の中小企業が見つかりやすい

---

### 方法3：LinkedIn（リンクトイン） ⭐⭐⭐⭐

**検索方法：**

1. LinkedIn の検索バーで以下を入力：
   - 「派遣 社長」
   - 「派遣会社 営業部長」
   - 「人材派遣 代表」

2. フィルター：
   - 業界：「人材サービス」
   - 場所：日本
   - 規模：1-10名、11-50名

3. 結果から個人プロフィールを確認 → DM送信

**メリット：**
- 決裁者に直接アプローチできる
- 共通の知り合い経由で紹介を得られる可能性
- 営業電話より丁寧な印象

**DMテンプレート：** メールテンプレート2「紹介経由」を流用

---

### 方法4：商工会議所・商工会の会員名簿 ⭐⭐⭐

**特徴：**
- 各地域の商工会議所が公開している会員リスト
- 地域密着・中小企業中心
- 商工会議所の信頼性が後ろ盾になる

**取得方法：**
1. 「○○商工会議所 会員名簿」で検索
2. 公開されているPDF/Web名簿から派遣会社を抽出
3. 商工会議所主催のセミナーに参加して名刺交換も◎

---

### 方法5：派遣業界団体 ⭐⭐⭐

| 団体名 | URL | 用途 |
|---|---|---|
| 日本人材派遣協会（JASSA） | https://www.jassa.or.jp/ | 大手中心、業界統計取得 |
| 全国求人情報協会 | https://www.zenkyukyo.or.jp/ | 中小派遣会社の動向把握 |
| 各都道府県の人材ビジネス協会 | 検索 | 地域コミュニティへの参加 |

**メリット：** 業界誌・セミナーで認知度を上げられる

## 📊 リスト管理テンプレート（CSVカラム）

スプレッドシートで以下のカラムで管理すると追跡しやすい：

```
| 番号 | 会社名 | 所在地 | 電話 | URL | 業種 | 推定スタッフ数 |
| 担当者名 | 役職 | メール |
| 初回コンタクト日 | 接触方法 | 反応 |
| 次回アクション日 | アクション内容 |
| 状態 | 備考 |
```

### 状態フェーズの定義

| 状態 | 意味 |
|---|---|
| 未接触 | リストアップ済み、未連絡 |
| 接触中 | 電話/メール送付済み、返信待ち |
| アポ取得 | デモ約束済み |
| デモ完了 | デモ実施済み、検討中 |
| トライアル中 | 30日無料試用中 |
| 成約 | 有料契約 |
| 失注 | 検討終了（理由をメモ） |
| 一時保留 | 半年後に再アプローチ |

### 1. クラウドツールを併用

### 2. テンプレ化で工数削減

- メール文面 → コピペ + 会社名を置換
- 電話スクリプト → 印刷して手元に
- デモPPT → 完成版を使い回し（営業資料.pptx）

### 3. 「断られても気にしない」マインドセット

- 30件かけて1件成約が平均
- 9割は断られる前提
- 1人の成約者が10人を呼ぶ（紹介）
