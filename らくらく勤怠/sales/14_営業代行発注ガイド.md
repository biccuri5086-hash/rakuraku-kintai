# 営業代行発注ガイド

**目標：月3〜10件のデモアポを成果報酬で獲得**

---

## 🎯 営業代行の選び方

### 重要な3つの軸

| 軸 | 推奨 | 避けるべき |
|---|---|---|
| **料金体系** | 成果報酬型（1アポ¥5,000〜15,000） | 着手金・月額固定（リスク高） |
| **専門領域** | SaaS・B2B経験あり | 訪問販売中心の代行会社 |
| **規模** | 個人〜小規模代行 | 大手（最低契約金額が高い） |

---

## 📋 おすすめ営業代行サービス3選

### 1. カクトク ⭐⭐⭐⭐⭐（推奨）

**URL：** https://kakutoku.jp/

**特徴：**
- フリーランス営業のマッチングプラットフォーム
- 1アポ¥5,000〜の成果報酬型を選べる
- SaaS営業の実績豊富
- 着手金なし、低リスクで開始可能

**料金例：**
- 1アポ：¥5,000〜¥15,000
- 1成約：¥30,000〜¥100,000（オプション）
- 月3アポなら最低¥15,000〜

---

### 2. ウィルオブ・ワーク（インサイドセールス）⭐⭐⭐⭐

**URL：** https://willof.jp/

**特徴：**
- SaaS企業向けインサイドセールス代行
- 月額固定型（¥20〜30万）
- ある程度の体制が整ってきたら検討

---

### 3. クラウドワークス・ランサーズ（フリーランス営業）⭐⭐⭐

**URL：** https://crowdworks.jp/

**特徴：**
- 個人副業の営業さんを直接募集
- 単価交渉が可能（1アポ¥3,000程度から）
- 質はピンキリ、選定が重要

## 📝 カクトクへの発注テンプレート

[カクトク](https://kakutoku.jp/) で「**営業依頼を出す**」をクリックして以下を入力：

### 案件タイトル

```
派遣会社向け SaaS のテレアポ営業（成果報酬型・月3アポから）
```

### サービス概要

```
ラクラク勤怠 - 派遣会社向け LINE 完結型勤怠管理 SaaS

【サービスの特徴】
・LINE で 1 タップ打刻、アプリ DL 不要
・コンディション報告で離職予兆を発見（業界唯一）
・150〜200円/人/月、初期費用ゼロ
・30日間無料トライアル
・本番稼働中（実顧客あり）

【ターゲット】
中小派遣会社（社員5〜30名、登録スタッフ50〜300名）
製造系・物流系・食品系・軽作業系を扱う会社

【決裁者】
社長または営業部長
```

### 依頼内容

```
リスト100社から、オンラインデモアポイントを取得していただきたいです。

【納品物】
・アポ取得時：日時、相手の氏名・役職・連絡先、ヒアリング内容
・週次レポート：架電数、接続数、断られた理由のまとめ

【トーク内容】
こちらでスクリプトをご用意しますので、その通りに進めていただければOK。
（営業スクリプト参照：「09_電話営業スクリプト.md」）

【目標】
月3〜10件のデモアポイント取得

【顧客リスト】
こちらで100社のリストをご用意します（クラウドワークスで作成済）

【支払い】
成果報酬型：1アポイント取得につき¥10,000
（取得アポイントが商談実施まで進んだ場合のみ支払い）

【その他】
・最低契約期間なし
・月3件以上の安定したアポイント取得を希望
・週1回30分の打ち合わせ（Zoom）でフィードバック
```

### 想定報酬

```
完全成果報酬型：1アポイント¥10,000
※月3件取得で¥30,000、月10件で¥100,000
```

### 募集条件

```
・SaaS・IT営業のテレアポ経験者
・B2B向け中小企業へのアプローチ経験あり
・人材業界への営業経験あれば尚可
・週20時間以上稼働可能な方優先
```

## ⚠️ 営業代行の落とし穴 7つ

### 1. 着手金型の代行は避ける
「月¥50,000で何アポ取れても同じ」だと、代行側のモチベが低い。**成果報酬型一択**。

### 2. 安すぎる単価は質が下がる
1アポ¥3,000以下だと、適当なアポが量産される（実は商談に進めない案件）。**¥10,000前後がベスト**。

### 3. リストは自分で用意
代行側に「リスト作成も込み」を依頼すると、質の悪いリストで失敗する。**リストは自分で発注**。

### 4. スクリプトを丸投げしない
代行側に任せきりにせず、自分の想いを込めたスクリプトを渡す。**サービスへの理解度が成約率を決める**。

### 5. 月1回はモニタリング架電
自分も時々アポ電話をする。**現場感を失わない**。

### 6. アポの「質」をフィードバック
取れたアポが「とりあえず話聞きたい」レベルなのか、「導入検討中」なのかを伝える。**質を上げてもらう**。

### 7. 契約解除条件を明記
3ヶ月試して効果出なければ別代行へ。**ダラダラ続けない**。

## 🚀 効果を最大化する3つのコツ

### 1. 「商談実施」まで成果報酬の対象にする

「アポ取得時点」だと、当日キャンセル多発のリスク。
「商談実施後」を成果対象にすれば、本気のアポが取れる。

### 2. インセンティブを段階的に

| アポ取得数/月 | 単価 |
|---|---|
| 1〜5件 | ¥10,000/件 |
| 6〜10件 | ¥12,000/件 |
| 11件以上 | ¥15,000/件 |

→ ボーナスで代行側のモチベ維持。

### 3. 成約ボーナスもセット

| 段階 | 報酬 |
|---|---|
| アポ取得 | ¥10,000 |
| デモ実施 | +¥5,000 |
| 30日トライアル開始 | +¥10,000 |
| 有料契約 | +¥30,000 |

→ アポの**質**が劇的に上がる。
