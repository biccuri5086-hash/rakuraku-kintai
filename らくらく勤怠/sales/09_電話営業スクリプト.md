# 電話営業スクリプト

中小派遣会社の社長・営業部長への電話アプローチ。

---

## 🎯 通話の目的

**1回の電話で目指すゴール：** 15分のオンラインデモアポイント獲得

❌ 電話で売り切ろうとしない（成約率が下がる）
✅ 「興味を持ってもらえたら、まず無料デモ」のスタンス

---

## 📞 電話スクリプト（基本版）

### 開始（受付突破まで）

```
受付: 「はい、○○株式会社でございます」

あなた: 「お忙しいところ恐れ入ります、
ラクラク勤怠の小原と申します。
派遣スタッフ向けの勤怠管理サービスの件で、
人事ご担当の方をお願いできますでしょうか？」

受付: 「どのようなご用件でしょうか？」（と聞かれる場合）

あなた: 「派遣スタッフのLINEで打刻できる新しいサービスのご案内です。
30秒で概要をお伝えしたいので、ご担当者様をお願いできますでしょうか？」
```

**ポイント：**
- 「営業」「セールス」と言わない → 「ご案内」「ご紹介」
- 「30秒だけ」と時間を明示 → 受付の心理的負担を下げる
- 「LINEで」を強調 → 業界ワード（差別化要素）で興味喚起

---

### 担当者への取り次ぎ後（最初の15秒）

```
担当者: 「もしもし、○○です」

あなた: 「お忙しいところ恐れ入ります、ラクラク勤怠の小原と申します。
30秒だけお時間頂戴できますでしょうか？

派遣スタッフが LINE を開いてボタンを1回押すだけで打刻できる
SaaSのご案内をしておりまして、

紙タイムカードの集計負担や、スタッフの突然離職に
お困りの派遣会社様に多くご導入いただいております。

価格は150〜200円/人/月で、初期費用ゼロ、
30日間全機能無料でお試しいただけます。

15分のオンラインデモのお時間を頂戴することは可能でしょうか？」
```

**ポイント：**
- 自社名 → 用件 → 解決する課題 → 価格 → アポ希望、の順で40秒以内
- 数字を入れる（150-200円、30日、15分）→ 具体性で信頼感
- 「お試し」「無料」を早めに → 警戒心を下げる

---

### よくある断り文句への切り返し

#### 「忙しい」「興味ない」

```
そうですよね、大変失礼いたしました。

ただ1点だけお伺いしたいのですが、
御社では今、派遣スタッフの方々の勤怠管理は
どのような方法でされていますでしょうか？

（タイムカード/紙の場合）
→ 「実は、紙からLINEに切り替えるだけで月の集計時間が
   90%以上削減されたお客様の事例がございまして…」

（他社SaaSの場合）  
→ 「現在お使いのサービス名差し支えなければ
   お伺いできますでしょうか？  
   比較検討用の資料だけでもお送りさせていただけませんか？」
```

#### 「資料送って」

```
かしこまりました、本日中にメールでお送りいたします。
メールアドレスをお伺いしてもよろしいでしょうか？

（メール聞いた後）
資料お送りした後、来週〇曜日の午後あたりに
内容のご確認のお電話を差し上げてもよろしいでしょうか？
```

**ポイント：** 必ず次のアクションを約束する → 連絡が立ち消えしない

#### 「LINEだとセキュリティが心配」

```
ご懸念ごもっともです。
弊社サービスは、データベース・通信全て暗号化されており、
管理者画面には2要素認証も導入しております。
ISO27001相当のセキュリティ設計になっておりますので、
詳細はデモの際にご説明させてください。
```

#### 「もう他社と契約してる」

```
ありがとうございます。
お使いのツールの名前を伺ってもよろしいでしょうか？

（聞き出した後）
弊社の場合、〇〇社さんと違って
「LINEで完結」と「コンディション報告機能」が独自の強みで、
派遣スタッフの離職予兆を早期に発見できる点が
派遣業特化として評価いただいております。

契約更新時期の参考に、15分だけ
特徴を比較できる資料をお送りしてもよろしいでしょうか？
```

---

### アポ取得成功時

```
ありがとうございます！
それでは、〇月〇日（〇）の14時〜14時15分でお時間頂戴したく、
Zoom URLは前日にメールでお送りいたします。

念のため、参加されるご担当者様のお名前と
メールアドレスを頂戴できますでしょうか？

オンラインデモでは以下のような内容をお見せします：
1. スタッフのLINE画面（実際に打刻してみる）
2. 管理者ダッシュボード
3. コンディション報告機能と離職予兆アラート

御社のスタッフ規模に合わせた料金シミュレーションも
ご用意しておきますので、現時点の登録スタッフ数も
ヒアリングさせてください。
```

## 📊 通話の評価指標

1日あたりの目安：

| 指標 | 目標 | 内容 |
|---|---|---|
| 架電数 | 30件 | 効率重視で時間内に多く |
| 担当者接続率 | 30% (9件) | 受付突破できた数 |
| アポ取得率 | 15% (4-5件) | 担当者と話せた中で |
| デモ参加率 | 60% (3件) | アポ取得した中で参加 |
| 成約率 | 30% (1件) | デモ参加した中で成約 |

→ **30件架電で1社獲得が目安**

## 🕐 電話をかける時間帯

| 時間帯 | 評価 | 理由 |
|---|---|---|
| 9:00-10:30 | ★★★★★ | 始業直後、まだ忙しくない |
| 10:30-12:00 | ★★★ | 業務集中時間、断られやすい |
| 12:00-13:30 | ★ | 昼休み、避ける |
| 13:30-15:30 | ★★★★ | 午後の余裕がある時間 |
| 15:30-17:00 | ★★★ | 一日の終わりで疲れている |
| 17:00以降 | ★ | 残業時間、迷惑になる |

**ベストタイム：火・水・木の 10:00-11:00 / 14:00-15:00**

---

## ⚠️ 絶対にやってはいけないこと

1. **嘘をつかない** → 「業界No.1」「無料です（実は有料）」など
2. **強引なクロージング** → 電話で契約まで持ち込もうとしない
3. **担当者を装わない** → 「○○の件で」と曖昧に取り次がせる
4. **断られても食い下がりすぎない** → 印象が悪化する
5. **個人情報を聞きすぎない** → 必要最小限のみ

## 💡 メモ：通話を録音する習慣

iPhone・Androidの通話録音アプリで自分の通話を録音し、後で聞き返してください。
1ヶ月続けると、自分の話し方の癖がわかり、トーク改善できます。

※ 相手に「録音している」旨は伝える方が誠実です（任意）
