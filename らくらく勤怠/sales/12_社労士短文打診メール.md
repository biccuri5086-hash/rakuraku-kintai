# 社労士事務所への打診メール集

**目標：1日に5〜10件送付 → 1〜2件の前向き返信を獲得**

---

## 🎯 戦略

長文の提携プログラム資料（既存PPT）は最初から送らない。

```
1通目：超短文メール（読みやすく、返信のハードルが低い）
   ↓
返信 or 興味あり
   ↓
2通目：詳細PPT送付＋15分Zoom面談打診
   ↓
面談で正式提案
```

この3段階で進めると返信率が3倍になります。

## 📋 社労士リストの作り方（無料・30分）

### 方法1：日本社会保険労務士連合会の検索（最速）

**URL：** https://www.shakaihokenroumushi.jp/Default.aspx

1. サイトを開く
2. 「**社労士検索**」をクリック
3. 都道府県・市区町村で絞り込み
4. 結果から事務所名・住所・電話番号を取得
5. 各事務所のHPからメールアドレスを取得

### 方法2：Google検索

```
「○○市 社会保険労務士」
「○○県 社労士事務所 メール」
「派遣業 社労士 ○○県」
```

### 方法3：商工会議所の会員名簿（地域密着）

地元商工会議所のHPから会員名簿を確認。
社労士事務所がリストアップされていることが多い。

## 🎯 ターゲットとすべき社労士の特徴

優先度高い順：

| 特徴 | 理由 |
|---|---|
| **派遣業を顧問先に持っている** | 即紹介可能 |
| **製造・物流系企業を顧問先に持つ** | 派遣会社との取引が多い |
| **HPに労務相談を打ち出している** | 営業に積極的 |
| **40〜50代の所長** | 新サービス導入に前向き |
| **地方の中堅事務所**（社員5〜20名） | 大手より話を聞いてくれやすい |

## ⚠️ NGメール文の例

以下は**送らないでください**：

❌ 件名「ご紹介プログラムのご案内」（営業メールが多すぎて埋もれる）
❌ 本文が長すぎる（読まれない）
❌ いきなりPDF添付（迷惑メール扱いされやすい）
❌ 「貴社」「貴方」など、社労士相手に使うべきでない表現
❌ 「お忙しいところ恐縮ですが」が連続している（くどい）

## 💡 返信率を上げる3つの小技

### 1. 件名に数字を入れる

```
× 「パートナー募集のお知らせ」
○ 「月¥3,000の継続収入：紹介パートナーのご相談」
```

### 2. メール冒頭に相手の事務所名

```
× 「お世話になります」
○ 「〇〇社会保険労務士事務所 〇〇先生」
```

→ 自分の名前が出ると人は読む。

### 3. 平日9時 or 火曜の午前中に送る

統計的に開封率が一番高い時間帯。
