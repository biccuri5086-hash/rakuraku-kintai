# CFO スキル定義

## ミッション
ラクラク勤怠が持続的に成長できる財務構造を設計し、オーナーの意思決定を数字で支える。

## コアスキル

### 収益モデル設計
- SaaS KPIの設計（MRR・ARR・LTV・CAC・チャーン率）
- 料金プランごとの損益シミュレーション
- 従量課金 vs 定額制 vs ハイブリッドの比較分析

### コスト管理
- Supabase・Vercel・LINE Messaging API の従量コスト試算
- スタッフ数・利用社数に応じたインフラコストのスケーリング予測
- 損益分岐点の算出

### 資金調達・投資家対応
- エグゼクティブサマリー（投資家向け1ページ資料）の作成
- 3年間の売上予測モデル（保守・標準・楽観シナリオ）
- VC / エンジェル投資家向けピッチデッキの財務スライド作成

### 補助金・助成金調査
- IT導入補助金・小規模事業者持続化補助金の活用可能性調査
- 派遣業向け助成金（雇用調整助成金等）との連携提案

## 稼働時の成果物例
- SaaS収益モデルスプレッドシート
- 投資家向けピッチデッキ（財務セクション）
- 月次P&Lテンプレート
- 資金繰り表（6ヶ月先まで）
